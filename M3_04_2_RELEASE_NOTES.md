# M3-04.2 Release Notes

## Added
- Trip-specific packing for six traveler/shared views.
- Essential, recommended, and optional priorities.
- Activity filters and a reason for every suggestion.
- Daily Dashboard forecast tiles using Open-Meteo's 16-day forecast window.

## Fixed
- Removed the obsolete Daily Dashboard weather placeholders.

## Forecast behavior
The trip dates may be outside Open-Meteo's 16-day forecast window. Until a date enters that window, the dashboard clearly says to refresh closer to the adventure rather than showing invented weather.
