# GitHub Upload Instructions

1. From the latest working `main`, create a branch such as `build-004-4c-experience-completion`.
2. Open **Add file → Upload files**.
3. Upload all root files from this package.
4. For the workflow, replace `.github/workflows/quality-checks.yml` at that exact path.
5. Commit with `Implement Build M3-04.4C Experience Completion`.
6. Confirm all GitHub Actions checks pass.
7. Preview the branch and complete every item in `VISUAL_QA_CHECKLIST.md`.
8. Merge only after the visual QA is successful.
9. After deployment, hard refresh and confirm Build Info reports `M3-04.4C`.
