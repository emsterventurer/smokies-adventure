# GitHub Upload Instructions

1. Open the `emsterventurer/smokies-adventure` repository.
2. Open the repository root.
3. Select **Add file → Upload files**.
4. Drag the contents of this ZIP into the upload area. Do not upload the outer ZIP itself.
5. Confirm `.github/workflows/quality-checks.yml` remains inside `.github/workflows/`.
6. Use commit message: `Implement Adventure Companion Build M3-04.4B`.
7. Commit to a review branch for the safest workflow, then merge after Actions pass.
8. After deployment, hard refresh the live page.
9. Open Build Info and confirm:
   - Version: `M3-04.4B`
   - Feature: Experience, Accessibility & Packing Celebration
   - Health: Ready / all diagnostics green
10. Test one packing checkbox, navigation, Daily Adventure, and diagnostics before marking the release complete.
