# Build Commit #004.1 — Dashboard Redesign & Adventure Intelligence

**Date:** July 20, 2026  
**Baseline:** Verified main after M3-04.2

## Included

- Removes the separate text timeline from every daily adventure.
- Makes the chronological Smart Stop Cards the visible itinerary.
- Replaces the six-tile dashboard with a lighter five-answer summary:
  - Leave around
  - First stop
  - Reservation
  - Total driving
  - Sunset
- Adds a day-level **Adventure Intelligence** card.
- Integrates the live trip forecast into Adventure Intelligence when available.
- Converts forecast data into a practical Adventure Outlook.
- Places the day's Plan B inside an expandable **Adaptive Adventure** section.
- Preserves flexible timing language already used by Smart Stop Cards, including “After lunch,” “After tasting,” and “Sunset.”
- Removes duplicated day-level Route/Parking, Foodie, Photos, and Plan B panels from the daily page.

## Scope note

Detailed parking, foodie, photo, and insider guidance will be attached to individual Smart Stop Cards in the next Smart Stop evolution build. Their removal here prevents the old duplicated page structure from remaining the primary experience.
